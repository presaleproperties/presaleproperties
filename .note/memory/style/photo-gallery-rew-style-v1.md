# Memory: style/photo-gallery-rew-style-v1
Updated: 2026-01-14

The site uses a REW-style 'REWPhotoGallery' component across all property detail pages (resale, presale, and assignments). Features include: 1) Preview with main image, photo counter, and thumbnail strip. 2) Full-screen modal with light gray background. 3) PHOTOS/VIRTUAL TOURS toggle tabs (when virtual tour URL available). 4) Scrollable 2-column grid layout for photos. 5) Clean header with back arrow and X close button. This replaces the previous dark-themed lightbox approach for a more premium, modern experience matching REW.ca aesthetics.
